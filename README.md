# Icosphere

This project dynamically generates and renders an icosphere using JavaScript and WebGL.

It was implemented as part of the Computer Graphics course at HSLU.

## Details

An icosphere is a sphere approximation created by subdividing the faces of an icosahedron.
This project generates the geometry algorithmically and renders it in WebGL. Lighting effects are based on the Phong Model.
